## Tips & Tricks

Simple Tips & Tricks sharing platform for engineers.

## Installation & Run

To install the package, you can use [Composer](https://getcomposer.org/):

```bash
composer install
```

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan migrate --seed
```

```bash
npm run dev
```

```bash
php artisan serve
```
